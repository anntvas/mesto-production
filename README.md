# Mesto

Интерактивный сервис Mesto для публикации карточек с изображениями.

## Функциональность

- Редактирование профиля пользователя
- Обновление аватара
- Добавление новых карточек
- Удаление собственных карточек
- Постановка и снятие лайков
- Просмотр изображений в полноэкранном режиме
- Валидация всех форм
- Получение и сохранение данных через API
- Публикация проекта на GitHub Pages

## Технологии

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Vite
- GitHub Pages

## Ссылка на проект

https://anntvas.github.io/mesto-production/

## Установка и запуск

Установка зависимостей:

bash npm install 

Запуск локального сервера:

bash npm run dev 

Сборка проекта:

bash npm run build 

Публикация проекта:

bash npm run deploy
